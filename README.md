# 📊 Matplotlib & Seaborn Basics

## 📖 Overview
This repository contains my hands-on practice while learning **Matplotlib** for data visualization in Python. It covers the fundamental plotting techniques that are essential for data analysis and machine learning.

## 🚀 What I Learned

- Importing Matplotlib, NumPy and Pandas
- Creating arrays using `numpy.linspace()`
- Basic Line Plot
- Adding Title and Axis Labels
- Creating Multiple Subplots
- Working with Figure Size and DPI
- Creating Multiple Axes
- Scatter Plot
- Histogram
- Box Plot
- Saving Figures using `plt.savefig()`
- Reading and Displaying Images using Matplotlib
